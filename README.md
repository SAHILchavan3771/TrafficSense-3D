# TrafficSense-3D
AI-powered Traffic Sign Classification System using CNN, Flask, and Three.js with interactive 3D visualization.

#Overview
TrafficSense 3D is an AI-powered traffic sign classification system that detects and classifies German traffic signs using a Convolutional Neural Network (CNN). The project integrates Flask for backend processing and Three.js for interactive 3D visualization.

#Features
Traffic Sign Detection & Classification
Deep Learning using CNN
Flask-based Backend API
Interactive 3D Visualization using Three.js
Real-time Prediction
User-friendly Web Interface

#Technologies Used
Python
Flask
TensorFlow / Keras
OpenCV
NumPy
HTML/CSS/JavaScript
Three.js

#Dataset

German Traffic Sign Recognition Benchmark (GTSRB)

43 Classes
50,000+ Images
Real-world traffic sign dataset
